# AI Interview Coach

AI Interview Coach is an open-source platform that helps students and job seekers prepare for technical interviews through AI-powered guidance.

## Features

### Resume Review

* AI-powered resume analysis
* Suggestions for improving ATS compatibility
* Identification of missing skills and keywords

### Mock Interviews

* Technical interview simulations
* Behavioral interview practice
* AI-generated feedback and improvement suggestions

### DSA Roadmaps

* Beginner, Intermediate, and Advanced learning paths
* Structured problem-solving progression
* Company-focused preparation tracks

### Company-Specific Preparation

* Google interview roadmap
* Microsoft interview roadmap
* Amazon interview roadmap
* Additional company tracks contributed by the community

## Tech Stack

### Backend

### Frontend

### AI Integration

* Gemini API

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/ai-interview-coach.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Backend

```bash
uvicorn main:app --reload
```
